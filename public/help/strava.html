Two-way sync

MyTracker syncs activities in both directions with Strava and Google Health. Import pulls workouts recorded on those platforms into MyTracker so they count toward your daily calorie target. Push sends workouts you log directly in MyTracker back to the external service so your activity history stays consistent everywhere.

Duplicate detection runs on every import: workouts that exist in both Strava and Google Health are matched by source ID so they are only counted once regardless of which service is the origin.

APIs used

Strava
Strava API v3

MyTracker reads activities from GET /api/v3/athlete/activities and writes back via POST /api/v3/uploads as a TCX file. This lets MyTracker include calorie data and heart rate (if logged) in the upload. OAuth uses the standard authorization code flow with scopes activity:read_all and activity:write. Strava reports total calories (including resting burn); MyTracker converts these to active-only calories on import by subtracting the BMR burn for the activity duration.

Google
Google Health API v4

Activities are read from and written to health.googleapis.com/v4/users/me/dataTypes/exercise/dataPoints. OAuth uses the PKCE flow with scopes googlehealth.activity_and_fitness.readonly and googlehealth.activity_and_fitness.writeonly. Token exchange runs through a Supabase Edge Function so client credentials are never exposed in the browser.

Connecting Strava

Go to Settings (gear icon) → Integrations & API Keys → Manage Integrations → Strava and tap Connect. You will be redirected to Strava's authorisation page. Once approved, MyTracker imports your last 100 activities on every load and keeps them in sync.

1
Open Settings → Integrations → Strava, tap Connect

You are sent to Strava's OAuth page to approve the connection. MyTracker requests read access to activities plus write access for pushing workouts back.

2
Activities are imported on every app load

Strava workouts appear in the Activities tab and count toward your daily calorie target automatically. Duplicate detection prevents the same workout from being counted twice if it also comes from Google Health.

3
Optionally push workouts back to Strava

Workouts you log directly in MyTracker can be pushed to Strava. Enable auto-push in Settings, or tap the push button on an individual workout card.

Calories on push

Activities are pushed to Strava as TCX file uploads. The TCX format lets MyTracker include the calorie count directly in the file, so Strava picks it up as-is. If you have a heart rate logged on the activity it is included too.

Custom Strava credentials

By default MyTracker uses a shared Strava app registration. If you want to use your own Strava API application (for example to remove the shared app name from your Strava activity feed), enable Use custom credentials in Settings and paste your own Client ID and Client Secret.

Google Health

Connect via Settings (gear icon) → Integrations & API Keys → Manage Integrations → Google Health. MyTracker pulls activities from Google Health (including workouts recorded by Android, Fitbit, and other connected apps). The same duplicate detection that applies to Strava activities applies here, so workouts that appear in both services are only counted once.

1
Open Settings → Integrations → Google Health, tap Connect

You are sent to Google's OAuth consent screen. MyTracker requests read and write access to your exercise data. Token exchange uses a Supabase Edge Function so your Google client credentials stay server-side.

2
Activities are imported on every app load

Google Health workouts appear in the Activities tab alongside Strava imports. Matching is done by source ID to prevent double-counting when both services hold the same workout.

3
Optionally push workouts to Google Health

Enable auto-push in Settings to have workouts you log in MyTracker written back to Google Health automatically. You can also push individual workouts from the workout card.

Cross-service push

When both Strava and Google Health are connected you can enable automatic cross-push between them:

Strava → Google
Auto-push Strava imports to Google Health

Each new activity imported from Strava is forwarded to Google Health automatically. Toggle in Settings → Integrations → Strava → Auto-push to Google Health.

Google → Strava
Auto-push Google Health imports to Strava

Each new activity imported from Google Health is forwarded to Strava automatically. Toggle in Settings → Integrations → Google Health → Auto-push to Strava.
